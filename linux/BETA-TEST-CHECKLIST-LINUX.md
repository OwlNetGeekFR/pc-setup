# Liste de test — PC Setup Linux Beta

À effectuer dans une machine virtuelle avant toute publication.

## Environnement

- [ ] L'application démarre depuis le menu Linux.
- [ ] La distribution et le gestionnaire de paquets sont reconnus.
- [ ] L'interface reste lisible en 1366 × 768 et en plein écran.
- [ ] Les liens officiels s'ouvrent dans le navigateur par défaut.

## Catalogue

- [ ] Les logiciels déjà installés portent la mention « Installé ».
- [ ] Une application native de la distribution s'installe correctement.
- [ ] Une application Flatpak s'installe correctement.
- [ ] Une demande d'autorisation administrateur apparaît uniquement si nécessaire.
- [ ] Une application installée peut être réparée par réinstallation.
- [ ] Une application de test peut être désinstallée.

## Maintenance

- [ ] La recherche des mises à jour se termine sans bloquer l'interface.
- [ ] La mise à jour du système respecte le gestionnaire de la distribution.
- [ ] Les applications Flatpak sont mises à jour.
- [ ] Les journaux sont créés dans `~/.local/share/pc-setup/logs`.

## Nettoyage

- [ ] L'analyse affiche une estimation avant suppression.
- [ ] Le nettoyage du cache utilisateur fonctionne.
- [ ] La Corbeille est vidée uniquement après confirmation.
- [ ] `Documents` et `Downloads` ne sont jamais supprimés.
- [ ] L'espace récupéré est affiché à la fin.

## Distributions minimales

- [ ] Ubuntu ou Linux Mint.
- [ ] Fedora.
- [ ] Arch Linux ou Manjaro.
