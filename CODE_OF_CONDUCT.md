# Code de conduite

## Notre engagement

OwlSetup souhaite offrir un espace accueillant, respectueux et constructif à toutes les personnes qui utilisent le projet ou y contribuent.

## Comportements attendus

- communiquer avec respect et patience ;
- accepter les critiques techniques constructives ;
- rester concentré sur l’amélioration du projet ;
- respecter la vie privée et la sécurité des utilisateurs ;
- reconnaître les contributions des autres.

Le harcèlement, les attaques personnelles, la discrimination, la publication de données privées et tout comportement perturbateur ne sont pas acceptés.

## Application

Les signalements concernant ce code de conduite doivent être adressés au propriétaire du dépôt. Le mainteneur peut modifier ou supprimer des contributions et limiter l’accès au projet lorsqu’un comportement enfreint ces règles.
