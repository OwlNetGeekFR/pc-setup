MODE D'EMPLOI

Installer en toute sécurité

Le script utilise uniquement winget et les fonctions Windows intégrées.

01

Composez votre sélection

Choisissez un profil ou ajoutez les applications une à une depuis le catalogue.

02

Générez le script

PC Setup crée localement un fichier PowerShell lisible, sans envoyer aucune donnée.

03

Exécutez en administrateur

Clic droit sur le fichier, puis « Exécuter avec PowerShell ». Confirmez la demande Windows.

04

Suivez l'installation

Chaque étape s'affiche dans la console. Un échec n'empêche pas les autres installations.

i
Avant de commencer

Créez un point de restauration et fermez vos applications. Certains composants peuvent demander un redémarrage.
