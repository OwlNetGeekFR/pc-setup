# Liste de test — PC Setup macOS Beta

- [ ] Compilation réussie sur Apple Silicon.
- [ ] Compilation réussie sur Mac Intel si disponible.
- [ ] L'application démarre depuis `~/Applications`.
- [ ] Le logo Apple apparaît dans la fenêtre et dans l'interface.
- [ ] Homebrew est détecté dans `/opt/homebrew` ou `/usr/local`.
- [ ] Une application Cask s'installe puis se désinstalle.
- [ ] Une formule Homebrew s'installe puis se désinstalle.
- [ ] Les logiciels déjà installés sont détectés.
- [ ] La recherche et l'installation des mises à jour fonctionnent.
- [ ] Réglages Système s'ouvre sur Mise à jour de logiciels.
- [ ] L'analyse de nettoyage affiche une estimation.
- [ ] Documents et Téléchargements restent protégés.
- [ ] Les journaux apparaissent dans `~/Library/Application Support/PC Setup/Logs`.
- [ ] Les liens officiels s'ouvrent dans le navigateur par défaut.
